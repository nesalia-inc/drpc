# Error to HTTP Status Mapping

## 5. Error to HTTP Status Mapping

This document defines how error codes are mapped to HTTP status codes.

| Error Code | HTTP Status |
|------------|-------------|
| `NOT_FOUND` | 404 |
| `VALIDATION_ERROR` | 400 |
| `UNAUTHORIZED` | 401 |
| `FORBIDDEN` | 403 |
| `CONFLICT` | 409 |
| `INTERNAL_ERROR` | 500 |
