# MDX Components

Default MDX components imported from `fumadocs-ui/mdx`.

## Usage

```mdx
import { Callout, Cards, ... } from 'fumadocs-ui/mdx';
```

## Available Components

Refer to individual component documentation:

- [Callout](../callout) - Alert boxes for notes, warnings, etc.
- [Cards](./cards) - Card layouts for links
- [Code Blocks](./code-block) - Syntax highlighted code
- [Steps](../steps) - Step-by-step guides
- [Tabs](../tabs) - Tabbed content
- [and more...](./)

## createRelativeLink

Function for creating relative links within documentation.

```tsx
import { createRelativeLink } from 'fumadocs-ui/mdx';
```
